Using factorial as an example.

Write a recursive solution that calculates a factorial. Keep in mind the following:

  • The factorial of 0 is equal to 1 // by definition
  • The factorial of 1 is equal to 1 // by definition
  • For all n greater than 1, n! = n * (n-1)!
