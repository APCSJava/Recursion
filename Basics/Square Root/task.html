Using recursion to calculate a square root

Write a recursive method that calculates the square root of a positive real number.

Approach:

  • Use two parameters, the original number and an initial guess, both positive real numbers.
  • Divide the original by the guessed value and compare the quotient to the guess parameter -- if they match, then the guess is actually the square root and you should return the guess.
  • If they do not match, use the quotient to refine the guessed value. A sufficient refinement is to use the average of the previous guess and the quotient.
  • Recursively invoke the method using successively refined guesses until guess and the quotient original/guess converge.

Sample:

  • root(9.0, 4.5)
  • 9.0 / 4.5 is 2.0 and 2.0 != 4.5 --> refine the guess
  • refined guess is the average of 2.0 and 4.5, 3.25
  • root(9.0, 3.25)
  • 9.0 / 3.25 is 2.7692 and 2.7692 != 3.25 --> refine the guess
  • refined guess is the average of 3.25 and 2.7692, 3.0096
  • root(9.0, 3.0096)
  • ... etcetera
