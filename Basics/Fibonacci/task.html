The Fibonacci Sequence

The Fibonacci sequence is a famous sequence that appears frequently in nature. This Wikipedia entry has information, as does this 3:19 YouTube video which explains how it relates to biologically efficient packing problems.

"SunFlower: the Fibonacci sequence, Golden Section" by lucapost is licensed under CC BY-NC-ND 2.0

In brief, the sequence begins {0, 1, 1, 2, 3, 5, ... } where each number after the first two can be calculated as the sum of the preceding two values.

Notice that the sixth term in the series (5, at index 5) can be determined by adding together the preceding two terms (fourth and fifth values) ie. 2 + 3 yields 5.

Implement a Fibonacci calculator using a recursive method. Given an index in the series, return the value at that index. Bear in mind the following:

  • The value at index 0 is 0 // a base case
  • The value at index 1 is 1 // another base case
  • For indices greater than 1,the value at index i is determined by summing the value at index i-1 with the value at index i-2
